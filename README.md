📒 Address Book (C Project)

📌 Overview

This project implements an Address Book system in C to manage contact details such as name, phone number, and email. It supports adding, searching, editing, and deleting contacts.



⚙️ Features
* Add new contacts
* Search contacts by name
* Edit existing contact details
* Delete contacts
* Store and retrieve data



🧠 Working
1. User selects an operation
2. Input is processed
3. Data is stored/updated in memory or file
4. Results are displayed to the user



💻 Requirements
* GCC Compiler
* C Programming



▶️ How to Run
1. Compile:

   ```bash
   gcc *.c -o address_book
   ```

2. Run:

   ```bash
   ./address_book
   ```

3. Choose operations from menu
